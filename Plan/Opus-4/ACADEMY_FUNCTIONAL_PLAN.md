# План функционала для веб-приложения онлайн академии

## Обзор проекта
Веб-приложение для IT-компании, которая запускает академию с тремя направлениями обучения:
1. **WordPress** (1 этап)
2. **Web Coding** (2 этап) 
3. **Shopify** (3 этап)

## Основные принципы
- Профессиональный dashboard для онлайн академии
- Универсальная платформа для всех направлений
- Персонализированный подход к обучению
- Интуитивный и удобный интерфейс

---

## ЭТАП 1: БАЗОВЫЙ ФУНКЦИОНАЛ (MVP)

### 1.1 Система аутентификации и авторизации
- **Регистрация студентов**
  - Форма регистрации с валидацией
  - Подтверждение email
  - Выбор направления обучения
  - Создание профиля студента

- **Авторизация**
  - Вход по email/паролю
  - Восстановление пароля
  - Запоминание сессии
  - Двухфакторная аутентификация (опционально)

- **Роли пользователей**
  - Студент
  - Преподаватель
  - Администратор
  - Модератор

### 1.2 Профили пользователей
- **Профиль студента**
  - Личная информация
  - Прогресс обучения
  - Достижения и сертификаты
  - Настройки уведомлений
  - Аватар и личные данные

- **Профиль преподавателя**
  - Биография и опыт
  - Специализация
  - Рейтинг и отзывы
  - Расписание занятий

### 1.3 Dashboard (главная панель)
- **Для студентов**
  - Текущий курс и прогресс
  - Ближайшие занятия
  - Задания к выполнению
  - Уведомления
  - Календарь обучения

- **Для преподавателей**
  - Активные курсы
  - Список студентов
  - Расписание занятий
  - Статистика успеваемости
