# Руководство по типам занятий в расписании

## Обзор

В системе добавлена поддержка различных типов занятий с цветовой индикацией для лучшей визуализации расписания.

## Доступные типы занятий

| Тип | Цвет | Описание |
|-----|------|----------|
| LESSON | Синий | Обычное занятие |
| EXAM | Красный | Экзамен |
| DEADLINE | Оранжевый | Дедлайн |
| MEETING | Зеленый | Встреча |
| WORKSHOP | Фиолетовый | Мастер-класс |
| SEMINAR | Индиго | Семинар |
| CONSULTATION | Желтый | Консультация |
| ANNOUNCEMENT | Розовый | Объявление |
| OTHER | Серый | Другое |

## Как использовать

### 1. Создание расписания группы

1. Перейдите в раздел "Группы" → выберите группу → "Расписание"
2. В разделе "Добавить расписание" выберите:
   - День недели
   - **Тип занятия** (новое поле)
   - Время начала и окончания
3. Нажмите "Добавить слот"

### 2. Просмотр расписания

- Каждое занятие отображается с цветным индикатором типа
- Рядом с временем показывается метка типа занятия
- Цвета помогают быстро различать типы занятий

### 3. Технические детали

- Типы занятий сохраняются в базе данных в таблице `GroupSchedule`
- Цвета и метки определены в файле `src/lib/event-types.ts`
- Поддерживается обратная совместимость - существующие записи получают тип "LESSON" по умолчанию

## Файлы изменений

- `src/lib/event-types.ts` - константы типов и цветов
- `src/app/admin/groups/[id]/schedule/page.tsx` - форма создания расписания
- `src/app/api/admin/groups/[id]/schedule/route.ts` - API для работы с расписанием
- `src/components/admin/GroupScheduleListView.tsx` - отображение списка занятий
- `prisma/schema.prisma` - схема базы данных
